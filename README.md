# ♟️ Chess Game(Pass and Play) Web Application

This project is a fully functional chess game built using React and TypeScript.

---

## 🚀 Live Demo

https://chess-ui-nine.vercel.app

---

## 📂 GitHub Repository

https://github.com/sreemani123/chess-ui

---

## ✨ Features

* ♟️ Complete chess rules implementation
* 🔄 Turn-based gameplay
* ⚠️ Check and Checkmate detection
* 🤝 Stalemate detection
* 🏰 Castling
* 🎯 En Passant
* ♛ Pawn Promotion
* ⏱️ Timer (5 min, 10 min, Unlimited)
* 🔁 Reset game

---

## 🧰 Tech Stack

* React.js
* TypeScript
* CSS
* React Hooks
* Git & GitHub
* Vercel

---

## ⚙️ Available Scripts

In the project directory, you can run:

### `npm start`

Runs the app in development mode.

### `npm run build`

Builds the app for production.

### `npm test`

Runs tests (if used).

---

## ⚙️ Installation

```bash
git clone https://github.com/sreemani123/chess-ui.git
cd chess-ui
npm install
npm start
```
